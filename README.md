# Todo List

## Запуск проекта

Установить зависимости:

```bash
npm install
```

Запустить приложение:

```bash
npm run dev
```

Запустить локальный API:

```bash
npm run server
```

Проверить сборку:

```bash
npm run build
```

Запустить тесты:

```bash
npm run test:run
```

Запустить e2e-тесты Cypress:

```bash
npm run dev
```

В отдельном терминале:

```bash
npm run test:e2e
```

Открыть Cypress App:

```bash
npm run test:e2e:open
```

Или запустить e2e-тесты одной коммандой:

```bash
npm run cypress
```

Проверить код линтером:

```bash
npm run lint
```
